Practice notes

Scope-safe reminders built into this version

  • The parent report uses non-diagnostic wording and focuses on concerns, strengths, and next steps.
  • The practitioner report separates observed features, caregiver-reported history, and referral recommendations.
  • The tool avoids final diagnostic wording and frames findings as screening indicators or reasons for onward referral.
  • The workflow leaves room for inclusion of standardised measures, but you should only use instruments currently classified and certified for use under the applicable HPCSA framework.
  • Data export and local browser backups are included so you are not dependent on Supabase only.
